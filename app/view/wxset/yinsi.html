用户隐私保护指引设置
1.配置后，需重新提交代码审核，审核通过且重新发布上线后才会在小程序端生效。
2.提交后还未重新发布上线前，再次打开看不到上次的配置为正常情况，上次的配置还未生效，接口读取的是已生效的配置，审核通过且重新发布上线后才会生效。
3.如果驳回，可以查看小程序审核驳回原因进行调整。
收集信息用途说明
{foreach $privacyArr as $k=>$item}
{if $k == 'UserInfo'}
例如 辨别用户身份
{elseif $k == 'Address'}
例如 地址导航
{elseif $k == 'PhoneNumber'}
例如 发送短信通知
{elseif $k == 'MessageFile'}
例如 上传评价图片等
{elseif $k == 'Location'}
例如 推荐最近的自提点
{elseif $k == 'Album'}
例如 评价晒单
{elseif $k == 'AlbumWriteOnly'}
例如 保存商品图片和海报
{elseif $k == 'ChooseLocation'}
例如 选择位置作为收货地址
{elseif $k == 'Clipboard'}
例如 复制订单号
{elseif $k == 'Contact'}
例如 保存联系方式
{else}
选填
{/if}
{/foreach} 信息收集方联系方式
邮箱、手机号、QQ、微信号 4种联系方式至少要填一种
例如 弹窗、公告或其他方式
境外主体的小程序必须选择地区，否则小程序审核会被驳回。
